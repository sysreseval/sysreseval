## Environment Variables

| Variable | Used by | Description |
|----------|---------|-------------|
| `USER_USERNAME` | sre CLI (with `--user`) | Student login name |
| `SRE_PUB_DIR` | sre CLI, GUI, tests | Override `/var/lib/sre` |
| `SRE_WRAPPER` | params.py | Override the sre-wrapper path |
| `LANG` | sre CLI | If set to a French locale, loads French help strings |

## Locale / Translations

The `sre` CLI uses `gettext` for its help strings. If `LANG` is set to a French locale (`fr_FR.UTF-8`, `fr`, etc.), the French catalog is loaded from `locale/fr/LC_MESSAGES/sre.mo` (dev) or `/opt/sre/locale/fr/LC_MESSAGES/sre.mo` (production). To recompile after editing `sre.po`:

```bash
make translations
```

The `sysreseval` GUI uses Qt translations (`.ts` / `.qm`). Recompile with `make translations`.
